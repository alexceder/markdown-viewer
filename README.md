# Chrome Markdown Extension

View your markdowns in the browser with GitHub flavoured style.

## Usage

Extension not yet in store. For now:

1. Clone
2. ``bower install``
3. Open ``chrome://extensions`` in Chrome.
4. Enable developer mode.
5. Load the package.
